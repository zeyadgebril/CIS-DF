# 🗄️ CIS-DF Certification Roadmap
A structured, self-paced roadmap to help you prepare for and obtain the **ServiceNow Certified Implementation Specialist – Data Foundations (CIS-DF)** certification — covering study resources, exam objectives, tools, and tips gathered along the way.

---

## 📌 About This Project
This repository documents my personal journey toward achieving the **CIS-DF** certification. It includes a step-by-step roadmap, curated learning materials, notes, and progress tracking — all organized to make the path as clear and actionable as possible.

The CIS-DF exam validates your ability to implement and configure **CMDB**, **CSDM**, and the broader Data Foundations layer within the ServiceNow platform.

Whether you're just starting out or already deep in your studies, feel free to use this as a reference or fork it for your own journey.

---

## 🎯 Exam Overview

| Detail | Info |
|---|---|
| **Full Name** | Certified Implementation Specialist – Data Foundations |
| **Prerequisite** | ServiceNow Certified System Administrator (CSA) |
| **Exam Format** | 75 multiple-choice questions, Matching |
| **Duration** | 90 minutes |
| **Passing Score** | 70% |
| **Delivery** | Online proctored via Kryterion |

## 🗺️ Roadmap

### Phase 1 — Prerequisites
- [ ] Hold an active **CSA** certification
- [ ] Familiarize yourself with ITIL concepts (Incident, Change, CMDB basics)
- [ ] Complete the **ServiceNow Fundamentals** course on Now Learning

### Phase 2 — Core Study
- [ ] CMDB Fundamentals (CI Classes, relationships, IRE)
- [ ] Identification & Reconciliation Engine (IRE) deep dive
- [ ] Reconciliation Rules & Data Manager
- [ ] CMDB Health dashboard & scoring
- [ ] Dynamic CI Groups
- [ ] Service Mapping concepts

### Phase 3 — CSDM Framework
- [ ] CSDM layers: Foundation, Business, Application, Technology
- [ ] Business Applications vs Application Services vs Technical Services
- [ ] Service Offerings & Service Commitments
- [ ] CMDB Governance best practices

### Phase 4 — Hands-on Practice
- [ ] Spin up a **Personal Developer Instance (PDI)** on developer.servicenow.com
- [ ] Configure CI Classes and relationships
- [ ] Run IRE simulations
- [ ] Build and test Dynamic CI Groups
- [ ] Map an Application Service end-to-end

### Phase 5 — Exam Prep
- [ ] Review the official **CIS-DF Exam Blueprint**
- [ ] Complete all mandatory Now Learning paths for CIS-DF
- [ ] Take practice questions & mock exams
- [ ] Review weak areas via PDI
- [ ] Schedule exam on **Webassessor (Kryterion)**

## 🏅 Certification

Upon passing, the badge is issued via **Credly** and can be verified publicly.

- **Verified by:** ServiceNow
- **Issued via:** Credly
- **Prerequisite Badge:** CSA

---

## 📝 Notes & Tips

- The IRE and Reconciliation Rules section carries heavy weight — spend extra time here.
- Understand the **CSDM v3 framework** layers thoroughly; several questions test conceptual mapping.
- Use your PDI to practice CMDB Health scoring — seeing the dashboard hands-on helps it stick.
- The exam is scenario-based, not just definitions — read each question carefully.
