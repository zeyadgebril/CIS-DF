This repository documents my personal journey toward achieving the CSA certification. It includes a step-by-step roadmap, curated learning materials, notes, and progress tracking — all organized to make the path as clear and actionable as possible.

Whether you're just starting out or already deep in your studies, feel free to use this as a reference or fork it for your own journey.
